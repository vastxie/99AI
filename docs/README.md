# 关于 99AI

99AI 是一个综合性的 AI Web 应用平台，旨在为用户提供一个集成化、易于部署的人工智能服务站点。参考[ChatGPT 官网](https://chat.openai.com)的理念，将多种 AI 技术集成于一个单一的平台中，从而提供了一个全方位的 AI 服务体验，包括但不限于对话、绘画、语音处理等功能。

## 系统特色

99AI 特别强调三个方面：**集成化**、**易部署性**、**开放性**。

### 集成化

99AI 将多种 AI 技术和模型集成在一个平台中，用户可以通过一个简单的界面访问到多样的 AI 服务，无需切换多个应用或网站。这种集成化的设计不仅提升了用户体验，也大大减少了使用门槛。

### 易部署性

基于现代化的 Web 技术栈，99AI 支持快速部署，无论是在虚拟主机、VPS 还是容器化环境中，用户都可以简单快捷地启动 99AI 服务，这使得即使没有专业技术背景的用户也能轻松拥有自己的 AI 服务站点。

### 开放性

99AI 致力于为用户提供一个开放、可自定义的平台。用户不仅可以根据自己的需求选择不同的 AI 模型，还可以定制化界面和功能，满足个性化的需求。此外，99AI 采用免授权、无后门的开源策略，保证了其开放性和安全性。

## 技术架构

99AI 采用了先进的 Web 技术构建其系统，前端使用 Vite + Vue3 框架搭配 TypeScript 进行开发，界面美观、响应迅速。后端则选择了 Node.js + NestJS，保证了服务的稳定性和扩展性。

## 应用场景

99AI 适用于多种场景，无论是个人爱好者想要探索 AI 技术，企业需要快速构建 AI 服务站点，还是开发者寻求一个可扩展的 AI 平台进行二次开发，99AI 都能提供满意的解决方案。它的易用性、高性能和灵活性使其成为当前市场上~~最具竞争力~~的 AI Web 应用之一。

## 发展愿景

99AI 不仅仅是一个产品，它更是一个不断进化的 AI 服务平台。未来，99AI 将继续融合更多前沿的 AI 技术，拓展其服务范围，提升用户体验，旨在成为领先的一站式 AI 服务平台。
